# Site delivery

1. Verify the handoff digest.
2. Have the readonly site manager return bounded ADR packets.
3. The root orchestrator launches site specialists in isolated roots.
4. Integrate and run `scripts/verify.sh`.
5. Ask operations excellence to review fresh evidence.
6. Return failures to the owning ADR; never bypass retries or self-approve.
7. A defective handoff stops site work. Returning to corporate `DESIGN` requires an
   explicit, digest-bound user rework authorization and a complete immutable snapshot.
